# SmackTranslate Plugin

A professional translation tool for WordPress that automates content localization.

## Tech Stack
- **Languages:** PHP, JavaScript
- **Frameworks/Tools:** WordPress API, CSS3

## Features
- Automated translation workflow for admin pages.
- Responsive design for backend controls.

## How to use
1. Clone this repository.
2. Upload the folder to your WordPress `/wp-content/plugins/` directory.
3. Activate via the WordPress Admin dashboard.
